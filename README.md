# 🚀 My Interactive Web Dashboard

This is a static website hosted on **GitHub Pages** that integrates Google Spreadsheets with custom interactive features for managing data and media.

## 🌐 Live Website
**View the project here:** [https://ycShea.github.io/MyWeb/](https://ycShea.github.io/MyWeb/)

---

## 🛠 Features
* **YouTube Integration:** A dedicated button to play specific video content.
* **Custom Messaging:** Input area with a "Large Font" (50px) display toggle.
* **Data Access:** Direct link to the master Google Spreadsheet.
* **Embedded Sheet:** A live, synced view of student data and survey results at the bottom of the page.

---

## 📂 How to Add or Update Files
To add new files (images, student data sheets, or exam forms) to this project:

1.  **Access the Repo:** Go to [https://github.com/ycShea/MyWeb](https://github.com/ycShea/MyWeb).
2.  **First-Time Setup:** If the repository is empty, click the link: **"creating a new file"**.
    <img width="100%" alt="createNewFile" src="https://github.com/user-attachments/assets/0c9c8aa5-bb4d-4c9b-8e3e-cadcdf93b7de" />
3.  **Add More Files:** Once active, click the **Add file** button at the top right of the file list.
4.  **Choose Method:**
    * **Create new file:** Write code (like `index.html`) directly in the browser.
    * **Upload files:** Drag and drop files from your Windows 10 computer.
5.  **Save Your Work (Commit):**
    * Click the green **Commit changes...** button.
    * Write a short note (e.g., *"Updated exam sheet"*).
    * Click **Commit changes** again to confirm.

## ⚙️ How to Change Website Content
To update the YouTube video or the Spreadsheet link inside the code:

1.  Click on `index.html`.
2.  Click the **Pencil Icon** (Edit this file).
3.  Update the `videoId` or the `iframe src` link.
4.  Click **Commit changes**. The live site updates automatically in ~60 seconds.

---

## 🚀 First-Time Launch Setup
If the website link is not active yet, follow these steps to "turn it on":

1.  Go to **Settings** (the gear icon ⚙️) at the top of the repository.
2.  On the left sidebar, click **Pages**.
3.  Under **Build and deployment** > **Branch**:
    * Click the dropdown that says `None`.
    * Select `main` (or `master`).
    * Ensure the folder is set to `/(root)`.
4.  Click **Save**.
5.  **Wait 1-2 minutes.** Refresh until you see: *"Your site is live at..."*
